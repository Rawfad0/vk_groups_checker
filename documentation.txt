vk_group_checker это программа для проверки присутствия пользователя вк в списке групп.

python 3.8

Зависимости:
requests

Команды:
check [mode] [url]      проверка пользователя (url) в списке, заданным модом;
                        выводит название и ссылку на соответствующие группы
extract [mode] [url]    добавление групп пользователя с открытой страницей (url) к списку, заданному модом
add [mode]              ручное добавление групп к списку, заданному модом
 [url]                  ввод групп по одному на строчку
 ...
 [url]
 end                    для завершения введите 'end'

Моды:
s                       short
f                       full

При завершении выполнения команды будет выведена точка на отдельной строке;
После этого можно будет ввести новую команду.
